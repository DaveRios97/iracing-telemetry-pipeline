# iRacing Telemetry & Analytics Pipeline

This repository contains a hybrid **Data Engineering (Modern Data Stack)** and **Advanced Analytics** solution built around high-frequency virtual motorsport telemetry from **iRacing**. 

It demonstrates the handling of real-time streaming data, distributed architectures, and advanced analytical proxy metrics (such as tire temperature differentials for understeer/oversteer diagnostics).
